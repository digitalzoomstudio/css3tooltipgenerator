Installation - top

All you have to do for installing on your site (ie. yoursite.com) is copy the contents of the /source folder to a folder (ie. yourfolder) on your site via ftp ( with filezilla ) or any other upload service. You can then access your contents via http://yoursite.com/yourfolder.

 

 

 

Integrating in your project - top

In order to install this component in your html page you need to:

  1. unpack the download package, open the source folder
  2. copy the folder /dzstooltip into your project folder
  3. open your html file and insert this inside your <head> tag
    <script src="dzstooltip/dzstooltip.js" type="text/javascript"></script>
    <link rel='stylesheet' type="text/css" href="dzstooltip/dzstooltip.css"/>
  4. insert this inside your <body> tag where you want the gallery to appear
    <span class="dzstooltip-con js"><span class="tooltip-indicator">insert anywhere</span><span class="dzstooltip talign-center arrow-bottom style-rounded color-dark-light dims-set transition-slidedown " style="width: 280px;"><span class="dzstooltip--inner">Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore.</span> </span></span>

     

  5. Customize freely using the customizations below
  6. All done!

 

Customise It - top

 

The optional js class

Normally, dzs tooltip can work without javascript , but javascript enables some extra features, like wrapping unstructured tooltips. For example, this structure ( the simplest ) is accepted, and will be corrected by javascript

<span class="dzstooltip-con js ">no structure<span class="dzstooltip talign-center arrow-bottom style-rounded color-dark-light dims-set transition-slidedown " style="width: 280px;">no structure</span></span>

 

This is optional and represents the container of the gallery and the preloader ( recommended ) . You insert video gallery items inside this container.

 

Orientation

Tooltip can be displayed on top, bottom, left, or right of the text. If you want it , at the bottom, you must add the arrow-top class to the dzstooltip container. So the class describes where the arrow would be located, not the position of the tooltip relative to the tooltip indicator.

 

<span class="dzstooltip-con js"><span class="tooltip-indicator">insert anywhere</span><span class="dzstooltip talign-center arrow-bottom style-rounded color-dark-light dims-set transition-slidedown " style="width: 280px;"><span class="dzstooltip--inner">Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore.</span> </span></span>

 

Other paramters you can add

  • aligment
    • talign-start - align tooltip at the start - top for arrow-right / arrow-left, and left for arrow-bottom / arrow-top
    • talign-center - center it
    • talign-end - align tooltip at the end
  • transition
    • transition-slidedown
    • transition-scaleup
    • transition-scaledown
  • color
    • color-dark-light
    • color-light-dark
    • color-magenta-yellow
  • style
    • style-default - plain tooltip container
    • style-rounded - rounded edges and shadow

What's the difference between jQuery(document).ready and jQuery(window).load?

When you want to manipulate pictures - ie: you depend on their size - like a ken burns slider or scroller does, it's best to use jQuery(window).load because that executes the code in it AFTER all the images and content has loaded. For everything else, there is jQuery(document).read -which only waits for the html to be parsed corectly & all elements formed.
